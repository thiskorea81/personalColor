# 🎨 AI 퍼스널 컬러 진단 (Personal Color Analyzer)

전면 카메라와 설문을 통해 당신의 퍼스널 컬러를 찾아주는 웹 애플리케이션입니다.

> **🔗 바로가기:** [https://fantastic-khapse-fd0e3a.netlify.app/](https://fantastic-khapse-fd0e3a.netlify.app/)

---

## 📝 프로젝트 소개

이 프로젝트는 별도의 서버나 앱 설치 없이, **웹 브라우저만으로 사용자의 퍼스널 컬러를 진단**할 수 있도록 제작되었습니다.

카메라를 통해 **피부톤 데이터**를 추출하고, 정밀도를 높이기 위한 **자가 진단 설문 결과**를 결합하여 최종적으로 **봄 웜톤, 여름 쿨톤, 가을 웜톤, 겨울 쿨톤** 중 하나를 추천해 줍니다.

## ✨ 주요 기능

### 1. 카메라 기반 피부톤 분석
* **실시간 얼굴 인식 가이드:** 사용자가 얼굴을 정확한 위치에 맞출 수 있도록 타원형 가이드를 제공합니다.
* **RGB 데이터 추출:** 촬영된 이미지의 중앙부(피부 영역) 픽셀을 분석하여 `Red`와 `Blue` 값을 비교, **웜톤(Yellow Base)**과 **쿨톤(Blue Base)** 경향성을 1차적으로 판단합니다.
* **미러링 모드:** 거울을 보는 것처럼 자연스러운 촬영 경험을 제공합니다.

### 2. 정밀 진단 퀴즈 (Quiz)
조명에 따라 달라질 수 있는 카메라 분석의 오차를 줄이기 위해 **5가지 핵심 질문**을 진행합니다.
* 액세서리 (골드 vs 실버)
* 햇볕 노출 반응
* 본연의 머리카락 색
* 어울리는 립 컬러
* 전체적인 이미지 (화사함 vs 또렷함)

### 3. 맞춤형 결과 리포트
* **나의 타입:** 4계절 유형 중 하나를 진단합니다.
* **추천 팔레트:** 해당 톤에 가장 잘 어울리는 **베스트 컬러 4가지**를 시각적으로 보여줍니다.
* **상세 설명:** 어울리는 스타일링 팁(Best)과 피해야 할 색상(Worst)을 안내합니다.

## 🛠 기술 스택 (Tech Stack)

이 웹페이지는 **Frontend-Only**로 구현되어 서버 없이 동작합니다.

| 분류 | 기술 | 설명 |
| :--- | :--- | :--- |
| **Framework** | `Vue.js 3` | CDN 방식, 반응형 UI 및 상태 관리 |
| **Styling** | `Tailwind CSS` | 유틸리티 퍼스트 디자인 시스템 |
| **Icons** | `Lucide` | 직관적인 아이콘 인터페이스 |
| **Deployment** | `Netlify` | 정적 사이트 배포 |

## 🚀 사용 방법

1. 웹페이지 접속 후 **'진단 시작하기'** 버튼을 클릭합니다.
2. 브라우저의 **카메라 권한**을 허용해 주세요.
3. 화면의 가이드라인에 얼굴을 맞추고 **촬영 버튼**을 누릅니다.
4. 피부톤 분석 결과를 확인한 뒤, 이어지는 **퀴즈**에 답합니다.
5. 최종적으로 분석된 **나의 퍼스널 컬러와 추천 색상**을 확인합니다.
